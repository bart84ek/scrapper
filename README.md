# where_is_my_ride

Scrap car auctions by brand and model.
- allegro.pl
- olx.pl
- otomoto.pl

## Run it

You need Oya task runner [Oya.sh](https://oya.sh/)

    $ oya run scrap nissan patrol

Or each one:

    $ oya run scrapOlx nissan patrol
    $ oya run scrapAllegro audi a4
    $ oya run scrapOtomoto toyota yaris
